# Mystic Siege

A top-down medieval fantasy survivor game inspired by Vampire Survivors and Brotato. Players choose a hero, survive escalating waves of enemies, collect XP orbs, level up, and pick upgrades — all in 15-30 minute sessions.

![Game Screenshot Placeholder](screenshot_placeholder.png)

## Installation

1. Clone the repository
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Running the Game

```
python main.py
```

## Controls

| Action | Key |
|--------|-----|
| Move | WASD or Arrows |
| Pause | ESC |
| Toggle Fullscreen | F11 |
| Toggle FPS Counter | F3 |
| Take Screenshot | F12 |
| Select Upgrade | 1, 2, 3 |

## Asset Credits

Placeholder assets are generated automatically. Real sprites can be added by placing PNG files in the appropriate `assets/sprites/` directories.

## Game Features

- Hero selection with unique passives
- 6 different weapon types with 5 upgrade levels each
- Progressive wave difficulty with enemy variety
- XP and leveling system
- Visual feedback for player actions
- Audio system with fallbacks
- Screenshot functionality (F12)
